\begin{DoxyParams}{Parameters}

\item FSAL\_\-RCP\_\-LOCAL\_\-TO\_\-FS: Copy the file from local path to the filesystem.
\item FSAL\_\-RCP\_\-LOCAL\_\-CREAT: Create the target local file if it doesn't exist.
